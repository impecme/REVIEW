# 第一章 电子材料概论

1. 结构材料和功能材料的区别？列举并简单解释功能电子材料。

| 类型     | 主要作用                 | 特点                                                                       | 常见用途                             |
| -------- | ------------------------ | -------------------------------------------------------------------------- | ------------------------------------ |
| 结构材料 | 承受压力、重力等机械作用 | 保持尺寸稳定，并具有一定强度、硬度、韧性等力学性能                         | 外壳、基片、框架、散热片、加固、封装 |
| 功能材料 | 实现特定功能             | 对光、电、磁、热、力、气氛等具有选择性响应，可实现不同形式能量或信号的转换 | 传感、光电转换、磁性器件、热电器件等 |

功能电子材料：主要应用不在于机械力学性能，而在于其特殊功能。例如太阳能电池材料可实现光能向电能的转换。

2. **晶体**的主要特征？概念包括**晶胞、晶格参数、晶系（七类）、晶向和晶面指数**等。理解典型**简单立方，体心立方 bcc，面心立方 fcc 和密排六方 hcp** 的结构特点。了解典型的钙钛矿结构，金刚石结构，尖晶石结构，闪锌矿结构和纤锌矿结构等。

**晶体主要特征：**
- 有规则的外形（自范性）
- 均匀性（一般情况晶体的性质与晶体的取样位置无关）
- 解离性
- 固定熔点
- 各向异性

**基本概念：**

| 概念     | 简要说明                                                                      |
| -------- | ----------------------------------------------------------------------------- |
| 晶胞     | 晶体结构中能代表整体周期性排列的最小重复单元                                  |
| 晶格参数 | 描述晶胞大小和形状，包括三个边长 $a、b、c$ 和三个夹角 $\alpha、\beta、\gamma$ |
| 晶面     | 不同原子组成的平面                                                            |
| 晶向     | 原子的线列方向，常用 $[uvw]$ 表示                                             |
| 晶面指数 | 表示某组平行晶面的符号，常用米勒指数 $(hkl)$ 表示                             |

**七大晶系：** 按照晶格参数及其之间的夹角，从对称性的角度进行分类。

| 晶系名称 | 晶格常数      | 夹角                                      |
| -------- | ------------- | ----------------------------------------- |
| 立方     | $a=b=c$       | $\alpha=\beta=\gamma=90^\circ$            |
| 正方     | $a=b\ne c$    | $\alpha=\beta=\gamma=90^\circ$            |
| 正交     | $a\ne b\ne c$ | $\alpha=\beta=\gamma=90^\circ$            |
| 六角     | $a=b\ne c$    | $\alpha=\beta=90^\circ,\gamma=120^\circ$  |
| 三角     | $a=b=c$       | $\alpha=\beta=\gamma\ne90^\circ$          |
| 单斜     | $a\ne b\ne c$ | $\alpha=\gamma=90^\circ,\beta\ne90^\circ$ |
| 三斜     | $a\ne b\ne c$ | $\alpha\ne\beta\ne\gamma\ne90^\circ$      |

**典型晶体结构：**

| 结构         | 结构特点                                                              |
| ------------ | --------------------------------------------------------------------- |
| 简单立方     | 原子位于立方体 8 个顶点，每个晶胞平均含 1 个原子，堆积效率较低        |
| 体心立方 bcc | 原子位于 8 个顶点和体心位置，每个晶胞平均含 2 个原子                  |
| 面心立方 fcc | 原子位于 8 个顶点和 6 个面心位置，每个晶胞平均含 4 个原子，堆积较紧密 |
| 密排六方 hcp | 原子按 ABAB 方式密排堆积，配位数高、堆积紧密                          |

**其他典型结构：**
- 钙钛矿结构：典型通式为 $ABO_3$，氧离子形成八面体结构。
- 金刚石结构：同种原子通过共价键形成四面体配位结构。
- 尖晶石结构：典型通式为 $AB_2O_4$。
- 闪锌矿结构：类似金刚石结构，但由两种不同原子交替占据晶格位置，属于立方结构。
- 纤锌矿结构：属于六方结构，也具有四面体配位。


3. 无机晶体的分类（按结合键分为**离子晶体、金属晶体、共价晶体、分子晶体**等），简单解释各结合键的特点。

| 类型     | 结合键特点                               | 性质特点                                             |
| -------- | ---------------------------------------- | ---------------------------------------------------- |
| 离子晶体 | 正、负离子通过静电吸引形成               | 结合力较强，熔点较高，通常硬而脆                     |
| 金属晶体 | 金属阳离子和自由电子形成金属键           | 导电性、导热性较好，具有金属光泽和延展性             |
| 共价晶体 | 原子之间通过共用电子对形成共价键         | 键合方向性强，饱和性强，结构稳定，通常硬度高、熔点高 |
| 分子晶体 | 分子通过范德华力、氢键等分子间作用力结合 | 结合力较弱，通常熔点低、硬度低                       |


4. 什么是**相变（多晶型转变）**？**有哪几个类型**？

相变：材料在温度、压力、电场、磁场等外界条件变化时，晶体中原子间的距离与原子振动情况发生了变动，致使原来的结构不再是稳定结构而转变成另一种结构的过程

多晶型：同一化学组成，却有不同结构的材料叫多晶型

多晶型转变：从某一晶体结构变成另一种结构的转变

常见类型：

| 类型       | 结构变化特点                                           | 转变特点                             |
| ---------- | ------------------------------------------------------ | ------------------------------------ |
| 位移型转变 | 相变通过原子移动，产生形变和键角的改变，但键并不会断裂 | 所需能量较低，转变较快，常具有可逆性 |
| 重构型转变 | 化学键断裂形成新键，导致新结构的产生                   | 所需能量较高，转变较慢，常有较高能垒 |

> 位移型侧重“原子位移、键不断裂”，重构型侧重“断键重组、结构重排”。

5. 什么是**固溶体**？有哪几个类型？什么是**金属间化合物**？

固溶体：固态条件下，一种组分溶解了其他组分形成的均匀晶态固体。

$$
\begin{aligned}
    \text{按固溶度分类}\quad
    &\begin{cases}
        \text{连续固溶体} & \text{两种组元可任意比例互溶} \\
        \text{有限固溶体} & \text{溶质只能在一定范围内溶入，超过固溶度会析出第二相}
    \end{cases} \\[1.2em]
    \text{按点阵位置分类}\quad
    &\begin{cases}
        \text{置换（替代）固溶体} & \text{溶质原子替代溶剂原子的位置} \\
        \text{间隙固溶体}         & \text{溶质原子进入晶格间隙} \\
        \text{缺位固溶体}         & \text{部分晶格位置空缺，成分偏离理想化学比}
    \end{cases}
\end{aligned}
$$

金属间化合物：两种或多种金属组分构成的具有特定组织结构的化合物，可存在金属键、离子键、共价键。

6. 实际晶体中的**缺陷**类型？**点缺陷（杂质、空位、填隙原子等）、线缺陷（刃型位错、螺型位错、混合位错等）、面缺陷（晶界、相界、层错、晶体表面等）、体缺陷等。**

$$
\begin{aligned}
    \text{点缺陷（零维）}\quad
    &\begin{cases}
        \text{空位}         & \text{晶格位置缺少原子} \\
        \text{填隙原子}     & \text{原子进入晶格间隙} \\
        \text{置换杂质}     & \text{外来原子替代原晶格原子} \\
        \text{肖特基缺陷}   & \text{正、负离子成对空位} \\
        \text{弗伦克尔缺陷} & \text{离子进入间隙并留下空位}
    \end{cases} \\[1.2em]

    \text{线缺陷（一维）}\quad
    &\begin{cases}
        \text{刃型位错} & \text{晶体中多出半原子面} \\
        \text{螺型位错} & \text{晶面绕位错线呈螺旋状错排} \\
        \text{混合位错} & \text{兼有刃型和螺型特征}
    \end{cases} \\[1.2em]

    \text{面缺陷（二维）}\quad
    &\begin{cases}
        \text{晶界}     & \text{不同晶粒之间的界面} \\
        \text{相界}     & \text{不同相之间的界面} \\
        \text{层错}     & \text{原子层堆垛顺序错误} \\
        \text{孪晶界}   & \text{两部分晶体呈镜面对称} \\
        \text{晶体表面} & \text{晶体与外界接触的界面}
    \end{cases} \\[1.2em]

    \text{体缺陷（三维）}\quad
    &\begin{cases}
        \text{孔洞}       & \text{晶体内部空腔} \\
        \text{夹杂物}     & \text{外来物质残留} \\
        \text{裂纹}       & \text{局部断裂缝隙} \\
        \text{沉淀相}     & \text{过饱和固溶体析出的新相} \\
        \text{第二相颗粒} & \text{基体中的另一种固相颗粒}
    \end{cases}

\end{aligned}
$$


7. 理解**单晶、多晶**、微晶、**纳米晶、非晶**、准晶等概念。（如**非晶和晶体的区别**等）

| 类型   | 结构特点                                                     |
| ------ | ------------------------------------------------------------ |
| 单晶   | 整个晶体内部晶格取向基本一致，长程有序，没有明显晶界         |
| 多晶   | 由许多取向不同的晶粒组成，晶粒之间存在晶界                   |
| 微晶   | 晶粒尺寸较小，但仍具有晶体结构                               |
| 纳米晶 | 晶粒尺寸达到纳米量级，晶界比例较高，表面效应明显             |
| 非晶   | 原子排列无长程周期性，没有固定晶格结构                       |
| 准晶   | 具有长程有序性，但不具有普通晶体的平移对称性，而是旋转对称性 |

8. 电子材料实际**表面**的特点？

表面：固体和气体的交界面。

界面：固体和液体，或不同固体相之间的交界面。

理想表面：表面原子与体内排列完全一致，只是周期性被打破。

实际表面：材料经过一般加工后，保持在常温常压下的表面，有时也可能处于真空或低温条件下。

| 表面类型     | 特点                                                             |
| ------------ | ---------------------------------------------------------------- |
| 未清洁表面   | 未经过特别清洗，含有较多污染物和吸附物                           |
| 清洁表面     | 经过清洗或处理，去除了大部分污染物，但仍可能有少量吸附层或氧化层 |
| 真空清洁表面 | 在真空中制备或处理，污染较少，较接近理想表面                     |

实际表面的特点：
- 表面外形和表面粗糙度：实际表面并非完全平整，存在形状误差、波纹度和表面粗糙度。
    - 形状误差：表面最高点与最低点之间的距离大于 10 mm。
    - 波纹度：表面起伏尺度约为 1~10 mm。
    - 表面粗糙度：表面微小起伏尺度小于 1 mm。
- 表面的组织：加工后的表面区在晶粒大小、结晶程度、应力畸变、显微组织等方面具有不均匀性。
    - 晶粒尺寸的变化：表层晶粒可能发生细化或长大。
    - 贝尔比层：机械抛光形成的光亮致密极薄表层，结构可能呈非晶态或严重畸变。
    - 一个经抛光和机械加工后的金属的表面区：打磨、抛光等加工会产生损伤和缺陷。
- 表面的成分：表面成分可能与内部不同，常出现氧化、吸附、污染或元素偏析。
- 工业表面：通常存在污染物、覆盖物、氧化层和加工损伤。

9. 简述先进电子材料的发展方向。（越丰富越好）

- 先进电子材料发展迅速：向高性能、高可靠性、小型化、集成化和多功能化方向发展。
- 有机电子材料应用不断扩大：具有柔性好、质量轻、可大面积制备等特点，可用于 OLED、柔性显示、有机太阳能电池等。
- 电子薄膜材料将成主流：薄膜材料便于集成化和微型化，是集成电路、传感器、光电器件中的重要材料形式。
- 半导体材料持续升级：从传统 Si 材料发展到宽禁带半导体，用于高频、高功率、高温等电子器件。
- 纳米电子材料受到重视：纳米尺度下表面效应、量子尺寸效应明显，可改善材料的电学、光学、磁学性能。
- 复合化和多功能化：通过多种材料复合，实现导电、介电、磁性、光电、热电等多种功能耦合。
- 柔性化和可穿戴化：材料需要具备柔性、轻薄、可弯曲等特点，适应柔性电子和可穿戴器件。
- 绿色化和低成本化：发展低能耗、环境友好、可回收或低成本制备的电子材料。

# 第二章 电子材料的分析和表征
1. **XRD 衍射峰包含哪些信息？**（从布拉格反射公式谈衍射角对应的晶面信息，衍射峰宽包含的晶格缺陷信息等）

X射线衍射基本原理：晶体内原子排列的对称性和周期性对于X射线来说是天然的光栅，属于光栅衍射，通过对衍射花样的研究和计算可以得到各种信息

XRD 的基本依据是布拉格反射公式：

$$
2d\sin\theta=n\lambda
$$

其中 $d$ 为晶面间距，$\theta$ 为衍射角，$\lambda$ 为入射 X 射线波长。不同晶面对应不同的 $d$ 值，因此会在不同的 $2\theta$ 位置出现衍射峰。

| 衍射峰信息 | 反映内容 |
| --- | --- |
| 峰位（$2\theta$） | 晶胞尺寸（点阵常数和晶面间距 $d$，可以唯一确定一个物相）、对称性、晶面取向 |
| 峰强 | 原子在晶胞中的位置，峰强大则结晶性好 |
| 峰宽 | 晶粒大小、点阵应变、缺陷，半峰宽大则晶粒小 |
| 峰形 | 可反映结晶质量、应力状态和样品均匀性等信息，对称性好说明缺陷少 |

> XRD 的峰位主要看“是什么晶面/什么物相”，峰宽主要看“晶粒大小和缺陷情况”。



2. 普通光学显微镜和电子显微镜的异同。初步理解透射电子显微镜（TEM）、扫描电子显微镜（SEM）的原理异同，工作模式等。

普通光学显微镜和电子显微镜的异同：

| 对比 | 普通光学显微镜 | 电子显微镜 |
| --- | --- | --- |
| 成像粒子 | 可见光 | 电子束 |
| 聚焦方式 | 玻璃透镜（折射原理） | 电磁透镜 |
| 分辨率 | 受可见光波长限制，分辨率较低 | 电子波长短，分辨率较高 |
| 观察对象 | 微米尺度组织形貌 | 微纳米尺度形貌、结构和缺陷 |
| 样品要求 | 制样较简单，可在空气中观察 | 制样要求高，通常需要真空环境 |

TEM 和 SEM 的区别：

| 对比 | TEM（透射电子显微镜） | SEM（扫描电子显微镜） |
| --- | --- | --- |
| 基本原理 | 电子束透过薄样品后成像 | 电子束扫描样品表面，收集二次电子、背散射电子、俄歇电子等信号成像 |
| 主要观察内容 | 内部结构、晶格、缺陷、衍射信息 | 表面形貌、颗粒形态、断口形貌等 |
| 样品要求 | 样品必须很薄，能让电子透过 | 样品表面导电性和稳定性较重要 |
| 图像特点 | 分辨率高，可观察内部细节 | 景深大，立体感强，适合观察表面形貌 |
| 常见工作模式 | 明场像、暗场像、选区电子衍射等 | 二次电子像、背散射电子像、俄歇电子像、能谱分析等 |

> TEM 看“内部结构和衍射”，SEM 看“表面形貌和立体形态”。
> 二次电子反映试样形貌，背散射电子反映原子序数，俄歇电子反映表面原子化学态

# 第三章~第五章
第 3-5 章关于制备工艺没有重点讲，需了解一定常规薄膜及厚膜的制备方法，如 PLD，CVD，磁控溅射等的概念。

薄膜材料：厚度较小、通常附着在基底上的材料层，常用于集成电路、传感器、光电器件等。

厚膜材料：厚度相对较大，通常通过浆料涂覆、印刷、烧结等方法制备，常用于厚膜电阻、电极、封装和陶瓷基电子元件等。

常见薄膜制备方法：

| 方法 | 基本概念 | 特点 |
| --- | --- | --- |
| PVD（物理气相沉积） | 通过物理方法使材料蒸发或溅射，并在基底表面沉积成膜 | 过程较洁净，适合制备金属、合金和部分化合物薄膜 |
| 蒸发镀膜 | 加热材料使其蒸发，蒸气在基底表面凝结成膜 | 原理简单，适合低熔点或易蒸发材料 |
| 溅射镀膜 | 利用高能粒子轰击靶材，使靶材原子溅出并沉积到基底上 | 薄膜附着力较好，成分控制较稳定 |
| 磁控溅射 | 在溅射基础上引入磁场，提高等离子体密度和沉积效率 | 应用广，可制备金属、氧化物、氮化物等薄膜 |
| PLD（脉冲激光沉积） | 用高能脉冲激光轰击靶材，使材料羽辉沉积到基底上 | 适合复杂氧化物薄膜，成分保持性较好 |
| CVD（化学气相沉积） | 气态反应物在基底表面发生化学反应并形成薄膜 | 膜层致密、覆盖性好，常用于半导体工艺 |

常见厚膜制备方法：

| 方法 | 基本概念 | 特点 |
| --- | --- | --- |
| 丝网印刷 | 将功能浆料通过丝网印到基底上，再经干燥和烧结形成膜层 | 工艺简单，适合批量制备厚膜电阻和电极 |
| 浆料涂覆 | 将含功能粉体的浆料涂覆在基底表面，干燥后烧结成膜 | 成本较低，膜层较厚 |
| 溶胶-凝胶法 | 由溶液形成溶胶，再转变为凝胶，经热处理得到材料层 | 组成均匀，适合氧化物材料 |

记忆要点：PVD 侧重“物理沉积”，CVD 侧重“化学反应成膜”；薄膜重在精细化和集成化，厚膜重在浆料、印刷和烧结。

> 第 6-12 章为各类不同应用领域的电子材料，重点在于是否理解其基本性质特点。其中第 8-11 章（半导体、磁性、光电材料为重中之重。）

# 第六章 导电材料和电阻材料

1. 导电率最高的金属（**银，铜**）。

导电率最高的金属是银，其次是铜。实际工程中常用铜作为导体材料，因为铜的导电率高、价格低于银、加工性能好。

百分电导率：以标准软铜的电导率作为 $100\%$，其他材料的电导率与其相比得到的相对值。

$$
\text{百分电导率}=\frac{\sigma}{\sigma_{\text{标准软铜}}}\times 100\%
$$

其中 $\sigma$ 为材料电导率。银的百分电导率略高于铜，铜约为 $100\%$，铝约为 $60\%$ 左右。

2. 列出几种常用的金属导体材料。
- 用的最多的是铜，其次是铝、铁等
- 铜合金：高导电性，高导热性，高强度，良好的伸长率
- 铝：良好的导电性，导热性，在空气中表面上可以生成氧化铝，密度小，易于加工成材，富于延展性，机械强度良好，广泛用作电极和引线材料


3. 在标准软铜中掺杂其他金属，对电导率有何影响？为什么？

在标准软铜中掺杂其他金属，一般会使电导率降低。

原因：
- 杂质原子进入铜晶格后，会破坏原有周期性势场，使电子运动受到散射。
- 杂质原子半径、电负性等与 Cu 不同，会引起晶格畸变，增加电子散射。
- 杂质含量越高，散射越强，电阻率增大，电导率下降

4. 电阻材料的**电阻与哪些因素**有关？什么是**电阻温度系数** α？金属材料的电阻温度系数有什么特点？（**金属材料的电阻一般随着温度的上升而升高，即正温度系数，其原因是什么？与常见半导体材料中存在的负电阻温度系数进行对比？**）

电阻与材料电阻率、长度和截面积有关：

$$
R=\rho\frac{L}{S}
$$

电阻温度系数 $\alpha$：温度每改变$1^\circ \mathrm{C}$时电阻的相对变化量

$$
\alpha=\frac{1}{R_0}\frac{\Delta R}{\Delta T}
$$


金属材料：一般为正温度系数，即温度升高，电阻增大。

原因：温度升高时，金属晶格振动增强，自由电子运动受到更强散射，电阻率增大。

金属和半导体的对比：

| 材料 | 温度升高后的电阻变化 | 原因 |
| --- | --- | --- |
| 金属 | 电阻增大，正温度系数 | 载流子数目变化不大，晶格振动增强导致电子散射增强 |
| 半导体 | 电阻通常减小，负温度系数 | 温度升高产生更多载流子，载流子浓度增加占主导 |

> 金属主要看“散射增强”，半导体主要看“载流子增多”。


5. 什么是体电阻和表面电阻？膜电阻（方阻）？定义

| 概念 | 定义 | 特点 |
| --- | --- | --- |
| 体电阻 | 电流通过材料内部体积时表现出的电阻 | 与材料体电阻率、长度和截面积有关 |
| 表面电阻 | 电流沿材料表面流动时表现出的电阻 | 与表面状态、污染、吸附层、湿度等有关 |
| 膜电阻（方阻） | 薄膜材料单位正方形区域两对边之间的电阻 | 与正方形边长无关，单位常用 $\Omega/\square$ |

方阻常用于描述薄膜电阻材料：

$$
R_s=\frac{\rho}{t}
$$

其中 $R_s$ 为方阻，$\rho$ 为材料电阻率，$t$ 为薄膜厚度。

对长为 $L$、宽为 $W$ 的矩形薄膜：

$$
R=R_s\frac{L}{W}
$$

> 体电阻看“内部导电”，表面电阻看“表面导电”，方阻看“薄膜单位正方形电阻”。


6. 列出几种常用的电阻材料以及主要应用场景，最好列举常用的合金材料或复合材料，并介绍其特点。

- 镍-铬系电阻合金：最常用的电阻材料，其有高的蒸汽压（易于真空蒸发和溅射）、耐热性良好、温度系数和热电势很小、稳定性高、可制作的组织范围宽，常用作小型精密片式电阻器和混合集成电路的薄膜电阻器
- 铬-硅电阻薄膜：片式元件和混合集成电路中常用的电阻材料，电阻率高、电阻温度系数小、稳定性好
- 金属-陶瓷电阻薄膜：金属和硅等氧化物绝缘体所组成的薄膜，电阻率高、耐温高

# 第七章 超导材料

1. **超导现象的定义。零电阻效应，完全抗磁性（迈斯纳效应）**

超导现象：某些材料在温度降低到临界温度 $T_C$ 以下时，电阻突然降为零，并表现出完全抗磁性的现象。

| 核心效应 | 含义 |
| --- | --- |
| 零电阻效应 | 超导态下电阻为零，电流可以无能量损耗地流动 |
| 完全抗磁性（迈斯纳效应） | 材料进入超导态后，会把内部磁场排斥出去，使磁感应强度趋近于零 |

> 注意：超导不仅是“电阻为零”，还必须具有迈斯纳效应，这是判断超导态的重要特征。


2. 低温超导和高温超导体的区分。列举典型的低温（最早发现的低于 4K 的汞 Hg 和一些典型的金属单质，还有金属间化合物 Nb3Ti、Nb3Ge 等）和高温超导体（$\mathrm{Y-Ba-Cu-O、LaOFeAs} 铁基超导体等）。可以稍微记一下记得越多越好。

| 类型 | 特点 | 典型材料 |
| --- | --- | --- |
| 低温超导体 | $T_C < 30^\circ \mathrm{C}$，液氦温度条件下工作，多为金属单质、合金或金属间化合物 | $\mathrm{Hg}$、$\mathrm{Pb}$、$\mathrm{Nb}$、$\mathrm{Nb-Ti}$、$\mathrm{Nb_3Sn}$、$\mathrm{Nb_3Ge}$ |
| 高温超导体 | 临界温度较高，液氮温度条件下工作，多为复杂氧化物或铁基超导体 | $\mathrm{Y-Ba-Cu-O}$、$\mathrm{La-Ba-Cu-O}$、$\mathrm{La-Sr-Cu-O}$ |

> 低温超导体以金属和合金为主，高温超导体以铜氧化物和铁基超导体为代表。


3. **BCS** 理论简单理解：**电子存在相干，形成“库珀对”，无晶格散射。**

BCS 理论是解释传统低温超导的微观理论，核心是电子通过晶格振动产生间接吸引，形成成对运动的库珀对。

| 关键词 | 简要理解 |
| --- | --- |
| 电子-声子相互作用 | 电子运动会引起晶格局部畸变，从而使两个电子之间产生有效吸引 |
| 库珀对 | 两个电子形成相干配对，整体运动具有稳定性 |
| 能隙 | 超导态与普通导电态之间存在能量差，使电子不易被散射破坏 |
| 零电阻 | 库珀对整体相干运动，不像普通电子那样被晶格散射而产生电阻 |

> BCS 理论可概括为“电子通过晶格作用形成库珀对，库珀对相干运动导致零电阻”。


4. **约瑟夫森效应**：超导电流流过约瑟夫森结发生隧道量子效应，是一种宏观量子相干现象，具有显著的非线性电阻特性，用于超导量子干涉器件（SQUID），特点是灵敏度极高。

> 约瑟夫森结：两个超导体之间夹一层薄绝缘层，称为约瑟夫森结


5. 超导体的三个临界参数：临界温度 $T_C$、临界磁场 $H_C$、临界电流密度 $J_C$。

| 临界参数 | 含义 |
| --- | --- |
| 临界温度 $T_C$ | 超导体由正常态转变为超导态的最高温度 |
| 临界磁场 $H_C$ | 超导态能够承受的最大外加磁场，超过后超导态被破坏 |
| 临界电流密度 $J_C$ | 超导体中能保持超导态的最大电流密度，超过后出现电阻或失去超导性 |

6. 第一类和第二类超导体的区别：（第二类超导体存在上下两个临界磁场，存在混合态，允许磁力线穿过超导体内部，钉扎效应，应用前景广阔。）

| 对比 | 第一类超导体 | 第二类超导体 |
| --- | --- | --- |
| 临界磁场 | 只有一个临界磁场 $H_C$ | 有两个临界磁场 $H_{C1}$ 和 $H_{C2}$ |
| 磁场行为 | 低于 $H_C$ 完全抗磁，高于 $H_C$ 失去超导性 | $H_{C1}$ 到 $H_{C2}$ 之间处于混合态 |
| 混合态 | 不存在明显混合态 | 存在混合态，允许部分磁力线进入超导体内部 |
| 材料类型 | 多为纯金属超导体 | 多为合金、化合物和高温超导体 |
| 应用特点 | 临界磁场较低，应用受限 | 临界磁场和临界电流较高，应用前景广阔 |

第二类超导体中的磁力线可以以磁通线形式进入材料内部。如果磁通线被缺陷或晶界固定，就形成钉扎效应，有利于提高临界电流密度。

> 第一类“一个临界磁场，直接失超”；第二类“两个临界磁场，中间有混合态”。

7. 列举超导体的应用领域，谈的越多越详细越好。

- 强电方面：
    - 超导输电：利用零电阻效应降低输电损耗。
    - 超导储能：利用超导线圈储存电磁能，响应速度快、效率高。
    - 超导电机和发电机：可提高功率密度和效率。
    - 超导磁体：可产生强磁场，用于大型科学装置和磁约束装置。
    - 磁悬浮交通：利用超导磁体实现悬浮和导向，减小摩擦。
- 弱电方面：
    - SQUID：利用约瑟夫森效应检测极微弱磁场，灵敏度极高。
    - 超导量子计算：利用超导量子比特进行量子信息处理。
    - 超导微波器件：用于低损耗滤波器、谐振器等。
    - 超导探测器：用于微弱光子、辐射或粒子信号探测。

> 强电应用看“零电阻和强磁场”，弱电应用看“约瑟夫森效应和高灵敏度”。

# 第八章 半导体材料（重点）
1. **半导体材料的定义**、特点和典型用途分类等。

半导体材料：导电能力介于导体和绝缘体之间，且电学性质容易受温度、光照、杂质、电场等因素调控的材料。

| 特点 | 简要说明 |
| --- | --- |
| 电导率适中 | 介于金属和绝缘体之间 |
| 可调控性强 | 可通过掺杂、温度、光照、电场改变导电性能 |
| 具有禁带 | 能带结构中存在禁带 |
| 可形成 p-n 结 | 是二极管、晶体管、太阳能电池等器件的基础 |

典型用途：集成电路、光电子器件、功率器件、传感器等。


2. **概念：本征和杂质半导体、载流子、能带理论相关的一系列概念（导带、价带、禁带、能级、施主受主能级，有效质量，平衡及非平衡载流子等）、p-n结、直接带隙和间接带隙、宽禁带和窄禁带半导体等。（属于本专业基本常识必须掌握）。**

| 概念 | 简要说明 |
| --- | --- |
| 本征半导体 | 纯净半导体，载流子主要来自本征激发 |
| 杂质半导体 | 通过掺杂改变导电性能，分为 n 型和 p 型 |
| 载流子 | 参与导电的电子和空穴 |
| 导带 | 电子进入后可参与导电的能带 |
| 价带 | 被价电子占据的能带 |
| 禁带 | 导带和价带之间不能存在电子能级的能量范围 |
| 施主能级 | 施主杂质引入，容易提供电子，形成 n 型半导体 |
| 受主能级 | 受主杂质引入，容易接受电子，形成 p 型半导体 |
| 有效质量 | 描述电子或空穴在晶体中运动惯性大小的等效质量 |
| 平衡载流子 | 热平衡状态下存在的载流子 |
| 非平衡载流子 | 光照、电注入等外界作用产生的额外载流子 |
| p-n 结 | p 型和 n 型半导体接触形成的结，具有内建电场和单向导电性 |
| 直接带隙 | 导带底和价带顶在同一动量位置，易发光 |
| 间接带隙 | 导带底和价带顶不在同一动量位置，发光效率较低 |
| 宽禁带半导体 | 禁带宽度较大，适合高温、高压、高频器件 |
| 窄禁带半导体 | 禁带宽度较小，常用于红外探测等领域 |


3. **半导体材料的光电性质（光吸收、光电导效应、光伏效应）；磁学性质（霍尔效应、磁阻效应）；热电性质（热电效应如塞贝克效应、帕尔贴效应、汤姆逊效应）对重点原理进行理解记忆。**

| 性质 | 效应 | 基本原理 |
| --- | --- | --- |
| 光电性质 | 光吸收 | 光子能量足够时，电子由价带跃迁到导带 |
| 光电性质 | 光电导效应 | 光照产生载流子，使电导率增大 |
| 光电性质 | 光伏效应 | 光生电子-空穴对在内建电场作用下分离，产生光生电压 |
| 磁学性质 | 霍尔效应 | 通电半导体在磁场中产生横向电压，可判断载流子类型 |
| 磁学性质 | 磁阻效应 | 外加磁场使材料电阻发生变化 |
| 热电性质 | 塞贝克效应 | 温差产生电压 |
| 热电性质 | 帕尔贴效应 | 电流通过接触处产生吸热或放热 |
| 热电性质 | 汤姆逊效应 | 电流通过有温度梯度的导体时伴随吸热或放热 |

> 光电效应看“光生载流子”，霍尔效应看“载流子类型”，热电效应看“热和电的相互转换”。


4. **三代半导体材料的概念**。典型第三代半导体材料列举及应用举例

| 代际 | 典型材料 | 主要特点 | 典型应用 |
| --- | --- | --- | --- |
| 第一代半导体 | Si、Ge | 工艺成熟 | 集成电路、二极管、晶体管 |
| 第二代半导体 | GaAs、InP | 电子迁移率高，光电性能好 | 高频器件、光通信、激光器 |
| 第三代半导体 | SiC、GaN、ZnO、金刚石 | 宽禁带，耐高温、耐高压、适合高频大功率 | 电力电子、射频器件、LED、激光器 |

第三代半导体重点：宽禁带、高击穿场强、高热导率，适合高温、高压、高频和高功率器件。

# 第十章 磁性材料（重点）
1. 物质磁性的来源？ **原子磁矩，电子自旋和轨道自旋**等概念的理解。

物质磁性主要来源于原子磁矩，而原子磁矩主要来自电子的自旋运动和轨道运动。

| 概念 | 简要说明 |
| --- | --- |
| 原子磁矩 | 原子内部电子运动产生的磁矩总和 |
| 电子自旋磁矩 | 由电子自旋产生，是许多磁性材料磁性的主要来源 |
| 电子轨道磁矩 | 由电子绕原子核运动产生 |
| 未成对电子 | 磁矩不能完全抵消，是强磁性产生的重要条件 |


2. **磁场强度，磁化强度和磁感应强度；磁化率和磁导率**等概念区别。

| 概念 | 符号 | 含义 |
| --- | --- | --- |
| 磁场强度 | $H$ | 外加磁场的强弱 |
| 磁化强度 | $M$ | 材料被磁化后单位体积内磁矩的大小 |
| 磁感应强度 | $B$ | 材料内部总磁场强度，反映磁场实际效果 |
| 磁化率 | $\chi$ | 表示材料被磁化的难易程度，常有 $M=\chi H$ |
| 磁导率 | $\mu$ | 表示材料导磁能力，常有 $B=\mu H$ |


3. **五种基本磁性？抗磁、顺磁、铁磁、反铁磁、亚铁磁，各自特点？**

| 类型 | 特点 |
| --- | --- |
| 抗磁性 | 外加磁场下产生与外场方向相反的弱磁化，磁化率为负 |
| 顺磁性 | 有未成对电子，外场下磁矩部分沿外场方向排列，磁性较弱 |
| 铁磁性 | 相邻磁矩平行排列，自发磁化强，具有磁畴和磁滞现象 |
| 反铁磁性 | 相邻磁矩反平行且大小相等，总磁矩近似为零 |
| 亚铁磁性 | 相邻磁矩反平行但大小不等，存在净磁矩 |


4. 强磁材料为什么往往包含 3d 过渡族金属（铁钴镍等）或 4f 族稀土元素？从未成对电子和铁磁性相互作用角度理解。

强磁材料往往含有未成对电子。Fe、Co、Ni 等 3d 过渡族金属具有未填满的 3d 电子壳层，稀土元素具有未填满的 4f 电子壳层，因此原子磁矩较大。

这些未成对电子的自旋磁矩在交换作用下趋向有序排列，从而形成较强的铁磁性或亚铁磁性。


5. **会画典型铁磁材料的磁化曲线和磁滞回线，并在图中标出如矫顽力、剩磁等参数。**

铁磁材料在外加磁场作用下，磁化强度逐渐增大并趋于饱和；撤去磁场后仍保留部分磁化，形成磁滞回线。

| 参数 | 含义 |
| --- | --- |
| 饱和磁化强度 $M_s$ | 磁畴基本沿外场方向排列后达到的最大磁化强度 |
| 剩磁 $M_r$ 或 $B_r$ | 外加磁场降为零后仍保留的磁化或磁感应强度 |
| 矫顽力 $H_c$ | 使剩磁降为零所需的反向磁场 |
| 磁滞损耗 | 与磁滞回线面积有关 |

> 软磁材料磁滞回线窄，硬磁材料磁滞回线宽。


6. 什么是**磁畴**？

磁畴：铁磁性或亚铁磁性材料内部，许多原子磁矩自发平行排列形成的小区域。每个磁畴内部磁矩方向一致，不同磁畴方向可能不同。

外加磁场作用下，磁畴会通过畴壁移动和磁矩转向而逐渐沿外场方向排列，使材料表现出宏观磁性。


7. 什么是**硬磁（永磁）和软磁**？根据矫顽力的大小及应用特点进行理解描述。

| 类型 | 特点 | 应用 |

# 第十一章 光电材料与热电材料（重点是光电材料）
1. **发光材料的主要类型**：光致发光、电致发光、阴极射线发光等。

| 类型 | 激发方式 | 典型应用 |
| --- | --- | --- |
| 光致发光 | 受光照激发后发光 | 荧光粉、发光标记、防伪材料 |
| 电致发光 | 在电场或电流作用下发光 | LED、OLED、显示器 |
| 阴极射线发光 | 电子束轰击材料后发光 | 阴极射线管、电子束分析 |


2. **电致发光**（主要是 LED 发光）机理的理解。

LED 发光机理：在 p-n 结正向偏置下，电子和空穴注入有源区并发生复合。复合释放的能量以光子形式发出，就产生发光。

发光颜色主要由半导体禁带宽度 $E_g$ 决定：

$$
h\nu \approx E_g
$$

直接带隙半导体更容易发光，发光效率较高。


3. 能够列举典型 LED 发光半导体材料，了解 OLED 等先进发光材料。

| 材料 | 典型发光范围或特点 |
| --- | --- |
| GaAs | 红外发光 |
| GaP | 红、绿等发光 |
| GaAsP | 红光、橙光、黄光 |
| AlGaInP | 红光、黄光高亮 LED |
| GaN、InGaN | 蓝光、绿光、白光 LED 的基础材料 |

OLED：有机电致发光材料，具有自发光、可柔性化、视角宽、响应快等特点，可用于显示和照明。


4. 了解光致发光材料（荧光和磷光材料）的概念和特点，典型应用场景。

| 类型 | 特点 | 应用 |
| --- | --- | --- |
| 荧光材料 | 激发停止后发光很快消失 | 荧光灯、显示、防伪、荧光探针 |
| 磷光材料 | 激发停止后仍能持续发光一段时间 | 夜光材料、安全标识、长余辉材料 |

光致发光过程：吸收光能，电子跃迁到激发态，再返回低能态并发出光。


5. 了解激光原理和典型材料。

激光的基本原理：通过受激辐射产生相干光。形成激光通常需要工作物质、粒子数反转和光学谐振腔。

| 典型材料 | 类型或应用 |
| --- | --- |
| 红宝石 | 固体激光材料 |
| Nd:YAG | 常用固体激光材料 |
| He-Ne | 气体激光器 |
| GaAs、InGaAsP | 半导体激光材料 |

激光特点：单色性好、方向性好、相干性好、亮度高。


6. 理解**光电转换材料（太阳能电池）的基本原理**（光生伏特效应）。

太阳能电池的基本原理是光生伏特效应。

基本过程：
- 光吸收：半导体吸收光子，产生电子-空穴对。
- 载流子分离：p-n 结内建电场使电子和空穴分离。
- 电流输出：电子和空穴经外电路定向运动，形成光生电流和光生电压。

常见太阳能电池材料：Si、GaAs、CdTe、CIGS、钙钛矿材料等。

> 太阳能电池就是“吸光产生载流子，内建电场分离载流子，外电路输出电能”。

# 第十二章 敏感材料
1. 敏感材料的概念定义，按功能分类：力声光电磁热气湿等。

敏感材料：对外界某种物理量、化学量或环境变化具有敏感响应，并能把这种变化转化为可检测信号的材料。

| 类型 | 响应对象 | 典型材料或器件 |
| --- | --- | --- |
| 力敏材料 | 压力、应力、应变 | 压阻材料、压电材料 |
| 声敏材料 | 声波、振动 | 压电陶瓷、声表面波材料 |
| 光敏材料 | 光照强度、波长 | 光敏电阻、光电二极管材料 |
| 电敏材料 | 电场、电压 | 压敏电阻材料 |
| 磁敏材料 | 磁场 | 霍尔材料、磁阻材料 |
| 热敏材料 | 温度 | 热敏电阻、热电偶材料 |
| 气敏材料 | 气体种类和浓度 | 金属氧化物气敏材料 |
| 湿敏材料 | 湿度 | 湿敏陶瓷、高分子湿敏材料 |


2. 典型传感器及敏感材料的应用举例。

| 传感器 | 敏感材料或原理 | 应用 |
| --- | --- | --- |
| 压力传感器 | 压阻材料、压电材料 | 压力检测、触觉传感 |
| 温度传感器 | 热敏电阻、热电偶材料 | 温控、电子设备保护 |
| 光传感器 | 光敏半导体材料 | 光强检测、自动控制、成像 |
| 磁传感器 | 霍尔材料、磁阻材料 | 电流检测、位置检测、磁场测量 |
| 气体传感器 | SnO2、ZnO 等气敏材料 | 有毒气体、可燃气体检测 |
| 湿度传感器 | 湿敏陶瓷或高分子材料 | 环境监测、家电控制 |
| 加速度传感器 | 压电材料、MEMS 结构 | 手机、汽车安全、振动检测 |

> 敏感材料的核心是“对外界刺激敏感，并把刺激转化为电学或可检测信号”。
