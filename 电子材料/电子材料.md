# 第一章 电子材料概论

1. 结构材料和功能材料的区别？列举并简单解释功能电子材料。

结构材料：能承受一定压力和重力，并能保持尺寸和大部分力学性质（强度、硬度、韧性等）稳定，主要用来制作外壳、基片、框架、散热片、加固、封装

功能材料：除了强度性能外，还有特殊功能，如能实现光、电、磁、热、力等不同形式的交互作用和转换作用，对外界环境具有灵敏的反应能力，对外界的光、电、磁、热、力、气氛等选择性的做出反应。主要应用不在于其机械力学性能，而是其特殊功能，如太阳能电池将光能转化为热能。

2. **晶体**的主要特征？概念包括**晶胞、晶格参数、晶系（七类）、晶向和晶面指数**等。理解典型**简单立方，体心立方 bcc，面心立方 fcc 和密排六方 hcp** 的结构特点。了解典型的钙钛矿结构，金刚石结构，尖晶石结构，闪锌矿结构和纤锌矿结构等。

晶体主要特征：
- 有规则的外形（自范性）
- 均匀性（一般情况晶体的性质与晶体的取样位置无关）
- 解离性
- 固定熔点
- 各向异性

概念：
- 晶胞：晶体结构中能代表整体周期性排列的最小重复单元。整个晶体可看作晶胞在三维空间中不断平移堆积而成。
- 晶格参（常）数：描述晶胞尺寸参数，包括三个边长 $a、b、c$ 。
- 晶系（七类）：按照晶格参数及其之间的夹角，从对称性的角度，可以分为七大晶系。

| 晶系名称 | 晶格常数 | 夹角 |
| --- | --- | --- |
| 立方 | $a=b=c$ | $\alpha=\beta=\gamma=90^\circ$ |
| 正方 | $a=b\ne c$ | $\alpha=\beta=\gamma=90^\circ$ |
| 正交 | $a\ne b\ne c$ | $\alpha=\beta=\gamma=90^\circ$ |
| 六角 | $a=b\ne c$ | $\alpha=\beta=90^\circ,\gamma=120^\circ$ |
| 三角 | $a=b=c$ | $\alpha=\beta=\gamma\ne90^\circ$ |
| 单斜 | $a\ne b\ne c$ | $\alpha=\gamma=90^\circ,\beta\ne90^\circ$ |
| 三斜 | $a\ne b\ne c$ | $\alpha\ne\beta\ne\gamma\ne90^\circ$ |

- 晶面：不同原子组成的平面叫晶面
- 晶向：原子的线列方向，常用 $[uvw]$ 表示，如 $[100]、[110]、[111]$。
- 晶面（密勒）指数：表示晶体中某组平行晶面的符号，常用米勒指数 $(hkl)$ 表示，如 $(100)、(110)、(111)$。

结构特点：
- 简单立方：原子位于立方体的 8 个顶点，每个晶胞平均含 1 个原子，结构简单但堆积效率较低。
- 体心立方bcc：原子位于 8 个顶点和体心位置，每个晶胞平均含 2 个原子，常见于 $\alpha$-Fe、Cr、W 等金属。
- 面心立方fcc：原子位于 8 个顶点和 6 个面心位置，每个晶胞平均含 4 个原子，堆积较紧密，常见于 Cu、Al、Ag、Au 等。
- 密排六方hcp：原子按 ABAB 方式密排堆积，配位数高、堆积紧密，常见于 Mg、Zn、Ti 等金属。

典型结构：
- 钙钛矿结构：典型通式为 $ABO_3$，A 位离子较大，B 位离子较小，氧离子形成八面体结构。许多铁电、压电、介电材料属于此类，如 $\mathrm{BaTiO_3}$。
- 金刚石结构：由同种原子通过共价键形成四面体配位结构，键合方向性强、硬度高，典型材料为金刚石、Si、Ge。
- 尖晶石结构：典型通式为 $AB_2O_4$，常见于铁氧体磁性材料，如 $\mathrm{Fe_3O_4}$、$\mathrm{MgAl_2O_4}$。
- 闪锌矿结构：类似金刚石结构，但由两种不同原子交替占据晶格位置，属于立方结构，典型材料有 ZnS、GaAs、InP。
- 纤锌矿结构：与闪锌矿成分相似但属于六方结构，也具有四面体配位，典型材料有 ZnO、GaN、CdS。


3. 无机晶体的分类（按结合键分为**离子晶体、金属晶体、共价晶体、分子晶体**等），简单解释各结合键的特点。

离子晶体：由正、负离子通过静电吸引形成，结合力较强、熔点较高，通常硬而脆。固态时离子不能自由移动，导电性差；熔融或溶于水后可导电。典型例子：NaCl、MgO、CaF2。

金属晶体：由金属阳离子和自由电子形成，电子可在晶体中自由运动，因此具有良好的导电性、导热性和金属光泽。金属键无明显方向性，所以金属通常具有较好的延展性。典型例子：Cu、Al、Fe、Ag。

共价晶体：原子之间通过共用电子对形成共价键，键合方向性强，结构稳定，通常硬度高、熔点高。多数共价晶体导电性较差，但 Si、Ge 等半导体具有重要电子功能。典型例子：金刚石、Si、Ge、SiC。

分子晶体：由分子通过范德华力、氢键等较弱的分子间作用力结合而成，结合力较弱，通常熔点低、硬度低、易挥发，导电性一般较差。典型例子：干冰、碘晶体、冰。

> 实际材料中常存在多种结合键共同作用，例如许多陶瓷和半导体同时具有离子键和共价键特征。


4. 什么是**相变（多晶型转变）**？**有哪几个类型**？

相变：当温度、压力、电场、磁场等外界条件变化时，晶体中原子间的距离与原子振动情况发生了变动，致使原来的结构不再是稳定结构而转变成另一种结构的过程

多晶型：同一化学组成，却有不同结构的材料叫多晶型

多晶型转变：从某一晶体结构变成另一种结构的转变

常见类型：
- 位移型转变：相变通过原子移动，产生形变和键角的改变，但键并不会断裂。一定温度下迅速发生，是可逆过程
- 重构型转变：化学键断裂形成新键，导致新结构的产生，比位移型转变需要更大的能量。特点是转变慢、需要较高能量

5. 什么是**固溶体**？有哪几个类型？什么是**金属间化合物**？

固溶体：固态条件下，一种组分溶解了其他组分形成的均匀晶态固体

按固溶度是否有限分类：
- 连续固溶体：两种组元可任意比例互溶（两组元晶体结构相同时才可能称为连续固溶体）
- 有限固溶体：溶质只能在一定范围内溶入，超过固溶度会析出第二相。

按溶质原子在点阵中的位置分类：
- 置换（替代）固溶体：溶质原子替代溶剂原子的位置
- 间隙固溶体：溶质原子进入晶格间隙
- 缺位固溶体：部分晶格位置空缺，其成分偏离理想化学比

金属间化合物：两种或多种金属组分构成的具有特定组织结构的化合物，存在金属键、离子键、共价键

6. 实际晶体中的**缺陷**类型？**点缺陷（杂质、空位、填隙原子等）、线缺陷（刃型位错、螺型位错、混合位错等）、面缺陷（晶界、相界、层错、晶体表面等）、体缺陷等。**

实际晶体并非完全理想，常存在不同维度的缺陷：

- 点缺陷：零维缺陷，发生在个别原子位置。{空位：晶格位置缺少原子；填隙原子：原子进入晶格间隙；置换杂质：外来原子替代原晶格原子；肖特基缺陷：离子晶体中正、负离子成对空位；弗伦克尔缺陷：离子离开原位进入间隙，同时留下空位。}
- 线缺陷：一维缺陷，主要是位错。{刃型位错：晶体中多出半原子面，位错线位于半原子面边缘；螺型位错：晶面绕位错线呈螺旋状错排；混合位错：同时具有刃型和螺型位错特征。}
- 面缺陷：二维缺陷，发生在晶体的界面或层面。{晶界：不同晶粒之间的界面；相界：不同相之间的界面；层错：原子层堆垛顺序发生错误；孪晶界：两部分晶体呈镜面对称关系；晶体表面：晶体与外界环境接触的界面。}
- 体缺陷：三维缺陷，占据一定体积。{孔洞：晶体内部的空腔；夹杂物：外来物质残留在晶体内；裂纹：局部断裂形成的缝隙；沉淀相：过饱和固溶体中析出的新相；第二相颗粒：基体中存在的另一种固相颗粒。}

缺陷不一定都是有害的。半导体中的掺杂原子属于人为引入的点缺陷，可调控载流子浓度和导电类型。
