# 纳米材料课本速通

## 一、表征技术（Characterization）
纳米材料的性能高度依赖其结构特征，因此精准的表征技术是核心：
### 1.1 表面与形貌分析：
1. 扫描电子显微镜（SEM）：观察表面形貌和尺寸分布。
2. 透射电子显微镜（TEM）：分析晶体结构、晶格间距和缺陷。
3. 原子力显微镜（AFM）：测量表面粗糙度及三维形貌。

### 1.2 结构分析：
1. X 射线衍射（XRD）：确定晶体结构、晶粒尺寸和相纯度。
2. 拉曼光谱（Raman）：分析化学键和材料应力（本质是测声子震动）。
   
   在碳材料中，D峰看缺陷，G峰看质量
   - G峰表示碳原子$sp^2$轨道伸缩运动，G峰尖锐、窄$\rightarrow$结晶度高、碳原子排列越整齐
   - D峰表示晶格中的缺陷状态，D峰越强$\rightarrow$缺陷越多
   - D/G峰强比值低$\rightarrow$材料质量极高，晶格近乎完美（如高质量石墨烯）
   - D/G峰强比值较高$\rightarrow$缺陷密度增加，可能是因为进行了表面改性、掺杂或者材料变碎了
   - D/G峰强比值极高$\rightarrow$材料可能已经趋向于无定形碳（结构非常混乱）

   在其他材料中
   - 峰的位置表征物相，且峰红移代表压应力导致的晶格收缩，峰的蓝移代表拉应力或温度升高导致的晶格膨胀
   - 半高宽表征结晶度，峰越窄结晶度越好，晶格排列越规整，峰越宽晶格缺陷多、应力分布不均或存在非晶相
   - 峰的强度与物质的浓度或厚度成正比
  
3. FTIR 傅里叶变换红外光谱：分析表面挂了什么官能团、化学键（本质是看化学键振动），不同峰的位置代表了不同的官能团、化学键

### 1.3 成分与化学状态分析：
1. X 射线光电子能谱（XPS）：确定表面元素组成和化学态。
2. 能量色散 X 射线光谱（EDX）：元素分布和成分分析。

### 1.4 其他：
1. 动态光散射（DLS）：测量纳米颗粒的流体力学尺寸。
2. 比表面积分析（BET）：评估多孔材料的孔隙结构。


## 二、性能评估（Performance Evaluation）
根据应用场景需系统评估材料的物理、化学和生物性能：

1. 电学性能：电导率、载流子迁移率（如用于柔性电子器件）。
2. 光学性能：吸收/发射光谱、量子产率（如量子点的发光效率）。
3. 机械性能：强度、韧性、弹性模量（如纳米复合材料的力学增强）。
4. 热性能：热导率、热稳定性（如纳米陶瓷的耐高温性）。
5. 磁性能：饱和磁化强度、矫顽力（如磁性纳米颗粒的靶向给药应用）。


## 三、稳定性与耐久性（Stability and Durability）
1. 化学稳定性：在特定环境（如酸/碱性、氧化性介质）下的抗腐蚀能力。
2. 热稳定性：高温下的结构稳定性（如纳米催化剂的热失活问题）。
3. 长期稳定性：储存和使用过程中是否发生团聚、氧化或相变。
4. 机械稳定性：在循环应力下的抗疲劳性（如纳米涂层在摩擦中的耐久性）。


## 四、 应用场景与兼容性（Application Compatibility）
生物相容性（针对生物医学应用）：
1. 细胞毒性测试（体外细胞实验、动物体内实验）。
2. 免疫原性评估（如用于药物递送的纳米颗粒是否引发免疫反应）。环境兼容性：
   1. 纳米材料在环境中的降解途径（如光降解、生物降解）。
   2. 对土壤、水体微生物的影响。

与基体材料的界面相容性（如纳米填料在聚合物基体中的分散性）。

# 必考点

## 一、XRD

### 1.1 XRD 工作原理：

XRD 即 X 射线衍射，当 X 射线照射到晶体时，晶体中的原子会使 X 射线发生**散射**。由于晶体中原子是周期性排列的，不同原子散射的 X 射线在空间会发生干涉。在某些特定方向上，散射波的相位相同，会产生相长干涉，形成衍射峰；在其他方向上，散射波的相位不同，会相互抵消。通过测量这些衍射峰的位置和强度，就可以分析出**晶体的结构和成分**。

### 1.2 XRD 图像反映的特征和信息：
1. **晶体结构信息：**
   
XRD 图谱中衍射峰的位置（2θ 角度）与晶体中原子的排列方式有关，通过布拉格方程 $n\lambda=2d\mathrm{sin}\theta$可以计算出**晶面间距**，进而确定**晶体的结构类型和晶胞参数**。
   - 峰位左移$\rightarrow 2\theta \downarrow \rightarrow d \uparrow \rightarrow$**掺杂大原子、受拉应力、热膨胀**
   - 峰位右移$\rightarrow 2\theta \uparrow \rightarrow d \downarrow \rightarrow$**掺杂小原子、受压应力、晶格缺陷收缩**
    

2. **物相分析信息：**
   
不同的晶体物质具有不同的 XRD 图谱，具有唯一性。将实验得到的 XRD 图谱与标准图谱库进行对比，就可以确定样品中存在的物相。

3. **结晶度信息：**
   
   - 结晶度好$\rightarrow$峰尖锐且强度高
   - 结晶度差$\rightarrow$峰宽化且强度较低

通过对衍射峰的宽度和强度进行分析，可以半定量地评估样品的结晶度。

4. **晶粒尺寸信息：**
   
谢乐公式：$D=\frac{K\lambda}{\beta \mathrm{cos}\theta}$（其中 D 为晶粒尺寸，K 为常数， 为衍射峰的半高宽， 为衍射角），可以由衍射峰的宽化程度估算出晶粒的平均尺寸。

### 1.3 XRD 图中反映的信息

1. **峰的位置：**
确定晶面间距$n\lambda=2d\mathrm{sin}\theta$，整个 XRD 光谱可以唯一确定一个物相

2. **峰强：**
结晶性好，某一晶面生长性好，几率大

3. **半峰宽：**
定性描述晶粒大小，半峰宽大则晶粒小

4. **峰的对称性：**
左右对称则说明晶粒比较完整，缺陷少

## 二、SEM：

### 2.1 工作原理：
   
   电子枪发射出的高能电子束，经电磁透镜聚焦后照射到样品表面。电子与样品相互作用，**产生二次电子、背散射电子等信号**。
   - 二次电子对样品表面形貌敏感
   - 背散射电子与样品原子序数有关。
   
   探测器收集这些信号并转化为电信号，再经处理后在显示屏上形成图像，从而获得样品表面的形貌信息。
### 2.2 可检测信息：
   
1.  表面形貌变化，如材料的断口、晶粒尺寸和形状、表面的缺陷与杂质等
2.  表面粗糙度
3.  结合EDS观察元素分布
4.  使用BSE（背散射电子）模式：原子序数（轻重），图案越亮，原子序数越大

### 2.3 其他特点：
   
1. 样品制备相对简单，一般只需将样品进行干燥、固定等处理
2. 对样品的损伤较小
3. 景深大，图像立体感强，可观察粗糙表面


## 三、TEM：
### 3.1 工作原理：
   
   电子枪发射的电子束经过聚光镜聚焦后，穿过超薄样品。由于样品对电子的**散射作用**，使得透过样品的电子束**强度分布发生变化**，携带了样品的结构信息。这些电子再经物镜、中间镜和投影镜等多级放大，最终在荧光屏或成像板上形成高分辨率的图像。
### 3.2 可检测信息：
   
1. 内部晶格结构，条纹代表晶面间距，如晶体的晶格结构、位错、层错等缺陷，还能观察到纳米尺度的颗粒、薄膜的厚度和结构等。
2. 衍射光斑
   - 单晶-->规则的衍射斑点（点阵）
    
      原子长程有序排列，特定布拉格衍射角下完全满足衍射条件，在倒易空间中形成排列规则、清晰、锐利的衍射斑点
   - 多晶-->同心环

      由许多随机取向的微小单晶组成，每一个小晶粒都会产生一套自己的衍射斑点，由于晶粒数量极大且取向各异，在同一晶面的衍射斑点集合在一起，围绕中心透射斑连接成了连续的同心衍射环
   - 非晶-->弥散晕环
      
      原子结构短程有序，内部结构相对混乱，电子束无法产生尖锐的布拉格衍射峰，发生漫散射，形成比较宽、边界模糊的光晕
3. 形貌（核壳结构、空心/实心）

### 3.3 其他特点：
   
1. 分辨率极高，可达到原子尺度。
2. 对样品要求苛刻，需制成厚度小于100nm 的超薄切片。
3. 操作复杂，对实验人员要求较高。



在纳米材料领域，表征与测试是指运用各种技术和方法对纳米材料的性质、结构等进行全面描述和测量的过程，以深入了解纳米材料的特性，为其性能优化和应用提供依据。

## 四、表征与测试的内容：
1. **结构表征：**
   
包括纳米材料的晶体结构、晶格参数、晶粒尺寸与形貌、表面与界面结构等。例如，通过 X 射线衍射（XRD）可确定纳米材料的晶体结构和晶格参数。

2. **尺寸与形貌分析：**
   
纳米材料的粒径大小、粒径分布以及颗粒的形状等。如透射电子显微镜（TEM）和扫描电子显微镜（SEM）可直观观察纳米颗粒的形貌和尺寸。

3. **表面性质：**
   
涉及表面化学成分、官能团、表面电荷、比表面积和孔隙结构等。比表面积和孔隙结构可通过氮气吸附 - 脱附实验进行测定。

4. **光学性质：**
   
如吸收光谱、发射光谱、荧光性质等。紫外 - 可见吸收光谱可用于研究纳米材料的光学吸收特性。

5. **电学性质：**
   
包括电导率、载流子浓度、迁移率等。四探针法可用于测量纳米材料的电导率。

6. **磁学性质：**
   
如磁化强度、磁滞回线、居里温度等。振动样品磁强计（VSM）可测量纳米材料的磁学性质。

## 五、常用的表征与测试方法：
1. **X 射线衍射（XRD）：**
   
当 X 射线照射晶体时，会发生衍射现象，通过分析衍射图谱可获得晶体结构信息。

2. **电子显微镜（EM）：**
   
如前面提到的 SEM 和 TEM，SEM 主要用于观察样品表面形貌，TEM 可用于观察内部精细结构和晶格图像。

3. **光谱分析：**
   
包括紫外 - 可见光谱、红外光谱、荧光光谱等。紫外 - 可见光谱可研究电子跃迁；红外光谱可分析分子的振动和转动能级，确定官能团；荧光光谱用于研究具有荧光特性的纳米材料。

4. **热分析：**
   
包括热重分析（TGA）和差示扫描量热法（DSC）。TGA 可研究纳米材料在加热过程中的质量变化，DSC 可测量样品的热流变化，用于分析相变、反应热等。

# 基础概念 

## 一、纳米材料定义：
1. 纳米材料是指在纳米量级（1~100nm）内调控物质结构制成具有特异功能的新材料，其三维尺寸中至少有一维小于 100nm，且性质不同于一般的块体材料。
2. 在性能上有突变或大幅度提高的材料把握“纳米材料”的概念，必须注意两点：
   1. 不是什么材料到了纳米尺度都有活性，如轻质碳酸钙就没有。
   2. 现在已用纳米粉体材料合成了具有特异性能的量子点、薄膜、多层膜、颗粒膜、块体材料和一维管、棒、线材料及复合材料，统称为纳米材料。
   
## 二、纳米材料的特性：
1. **表面与界面效应：**
纳米材料的比表面积大，表面原子数增多，导致表面能和表面张力增加，使纳米材料具有较高的化学活性和吸附能力。

2. **小尺寸效应：**
当颗粒尺寸进入纳米量级时，其声、光、电、磁等特性与宏观材料相比发生显著变化，如纳米金属颗粒的熔点远低于块状金属。

3. **量子尺寸效应：**
粒子尺寸下降到临界值时，费米能级附近的电子能级由块体中的连续能级分裂成离散能级的现象**或描述为半导体纳米颗粒的带隙变宽现象**，从而使纳米材料具有独特的光学、电学等性质，如纳米半导体材料的**吸收光谱出现蓝移现象。**

4. **宏观量子隧道效应：**
微观粒子具有贯穿势垒的能力，纳米材料中的电子等微观粒子也能表现出这种隧道效应，这对纳米器件的设计和应用具有重要意义。

## 三、纳米材料按空间尺度分类：
1. **零维纳米材料：**
在空间三维尺度均在纳米尺度的纳米材料。

2. **一维纳米材料：**
在空间有两维方向上处于纳米尺度，而第三维为宏观尺寸。

3. **二维纳米材料：**
在一维方向上尺寸被限制为纳米量级的层状结构，即在两个空间坐标上的延展。

4. **三维纳米材料：**
在三个空间坐标的延续，也就是块体材料，由最小构成单元为纳米结构的材料制成。

超微颗粒具有很高的活性，无机纳米粒子在空气中会吸附气体，并与气体发生反应。

10. 量子尺寸效应：粒子尺寸下降到临界值时，费米能级附近的电子能级由块体中的连续能级分裂成离散能级的现象/半导体纳米颗粒的带隙变宽现象
11. 表面效应：表面原子数与总原子数之比随纳米粒子的减小而大幅度增加，导致表面原子配位严重不足，具有多个悬挂键，表面能增大而容易与其他原子结合
   - 团聚现象
   - 氧化现象
   - 晶格收缩
12. 库伦阻塞效应：半导体纳米粒子位几十纳米时，电荷是量子化的，体系内充放电过程是不连续的，电子不能集体运输，只能一个一个单电子传输

   - $\mathrm{Li,Be}$能量太低导致探测不到
